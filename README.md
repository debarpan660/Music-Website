# 🎵 Music Website

A responsive Music Website with a custom music player built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🎵 Play / Pause
- ⏭ Next / Previous
- 🔀 Shuffle Mode
- 🔁 Repeat Mode
- ❤️ Favorite Songs
- 🔍 Search Songs
- 🎚 Progress Bar
- 🔊 Volume Control & Mute
- 💿 Rotating Album Cover
- ✨ Equalizer Animation
- ⌨ Keyboard Shortcuts
- 📱 Responsive Design
- 🎨 Glassmorphism UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
